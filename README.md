###How to use

- import serpent.py 

```
import serpent
```

- write some code

```
#!/usr/bin/python
import serpent

print "hello world"
```

- compile

```
python some.file.py
```

- Open the newly generated serpent file that's extrememly freaking long and totally wasting space but who cares it's a kickass
  snake wrapped around a kickass sword!!
